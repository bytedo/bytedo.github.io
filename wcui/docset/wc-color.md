## 取色器
> 常规的取色器, 类似`input[type=color]`。

```html
<wc-color value="#0099ff"></wc-color>
```

<wc-color value="#0099ff"></wc-color>


### 取值
> 通过`value`属性设置或读取数值。这个没啥可说的了。


### 完整APIs

|  属性  |  值类型  |   默认值   |     描述   |
|  :-:  |   :-:   |   :-:   |     -   |
|  value  |  String  |   无   |   支持十六进制颜色码及rgb, rgba, 如`#ffffff, rgb(0,0,0), rgba(255,0,0,.5)`   |
